ラベル付けプロジェクトは、Azure Machine Learning から管理します。 **[データのラベル付け]** ページを使用して、プロジェクトを管理します。

データが既に Azure Blob Storage 内にある場合は、ラベル付けプロジェクトを作成する前に、それをデータストアとして使用できるようにする必要があります。 データストアの使用例については、[初めての画像分類のラベル付けプロジェクトの作成に関するチュートリアル](../articles/machine-learning/tutorial-labeling.md)を参照してください。
