# <a name="quotas-for-azure-container-apps-preview"></a>Azure Container Apps プレビューのクォータ

Azure Container Apps プレビューには、次のクォータがあります。

| 機能 | 数量 |
|---|---|
| 環境 | 2 |
| 環境あたりのコンテナー アプリ | 20 |
| コンテナー アプリあたりのレプリカ | 25 |
| レプリカあたりのコア | 2 |
| 環境あたりのコア | 50 |
