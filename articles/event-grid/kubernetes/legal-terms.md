# <a name="legal-terms-and-third-party-notices"></a>法律条項とサードパーティに関する通知
この記事では、Kubernetes 上の Event Grid に関する法律条項とサードパーティに関する通知について説明します。 

## <a name="legal-terms"></a>法律条項 

Azure Arc を使用した Kubernetes 上の Event Grid の使用を既定する法律条項は、次のディレクトリにある Event Grid 演算子と Event Grid ブローカー コンテナーのファイルシステムにあります。 

Event Grid 演算子: ``/LICENSE``。

Event Grid ブローカー: ``/app/LICENSE``。


## <a name="third-party-notices"></a>サード パーティに関する通知

Azure Arc を使用した Kubernetes 上の Event Grid のサードパーティ ソフトウェアに関する通知を概説するドキュメントは、次のディレクトリにある Event Grid 演算子と Event Grid ブローカー コンテナーのファイルシステムにあります。

Event Grid 演算子: ``/NOTICE.txt``。

Event Grid ブローカー: ``/app/NOTICE.txt``。
