Azure サブスクリプション、課金、契約がなくても [Azure Cosmos DB を無料でお試し](https://azure.microsoft.com/try/cosmosdb/)いただけます。または、[Azure Cosmos DB Free レベルのアカウント](../optimize-dev-test.md#azure-cosmos-db-free-tier)を作成して、最初の 400 RU/秒と 5 GB のストレージを無料でご利用いただけます。 また、URI `https://localhost:8081` で [Azure Cosmos DB Emulator](../local-emulator.md) を使用することもできます。 エミュレーターで使用するキーについては、「[要求の認証](../local-emulator.md#authenticate-requests)」を参照してください。
