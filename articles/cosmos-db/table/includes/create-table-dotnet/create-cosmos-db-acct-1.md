Azure portal で次の操作を行います。

   1. Azure portal の上部の検索バーに「cosmos db」と入力します。
   1. 検索バーの下に表示されるメニューの **[サービス]** で、「**Azure Cosmos DB**」とラベル付けされた項目を選択します。
