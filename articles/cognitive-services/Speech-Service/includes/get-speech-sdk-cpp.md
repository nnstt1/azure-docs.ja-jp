#### <a name="c-nuget-package"></a>C++ NuGet パッケージ

C++ 音声SDKは **パッケージマネージャー** から以下の`Install-Package`コマンドでインストールができます。

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
