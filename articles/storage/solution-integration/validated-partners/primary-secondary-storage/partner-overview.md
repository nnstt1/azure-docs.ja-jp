# <a name="azure-storage-partners-for-primary-and-secondary-storage"></a>プライマリおよびセカンダリ ストレージの Azure Storage パートナー

この記事では、ネットワーク接続ストレージ (NAS) または記憶域ネットワーク (SAN) ソリューションを提供する Microsoft パートナー企業について説明します。 コスト効率の高い階層として Azure Storage を使用する、オンプレミス、Azure、またはハイブリッドのソリューションがあります。 これらのソリューションにより、お客様はどの環境でも同じソリューションを使用できるようになります。
